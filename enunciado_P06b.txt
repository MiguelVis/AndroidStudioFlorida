Versión Google Maps

Una vez que tenemos la longitud y latitud de nuestra posición, sería fantástico mostrarlo a través de un mapa en Google Maps.
Para ello seguiremos utilizando Google API Services y Google Maps

Pasos a realizar

Utilizaremos el sexto proyecto como base

Modificaremos el texto de Localización por dos posibles opciones:

 - Un fragment estático dentro de nuestro fragment de perfil
 - Un fragment que se cargaría de forma independiente a nuestro fragment de perfil

Generamos nuestra autorización para usar el Google Maps API (https://developers.google.com/maps/documentation/android-api/signup, https://developers.google.com/maps/documentation/android-api/start)

Utilizamos el ayudante de Android Studio para incorporar el google el google_maps_api y resto de código

Incorporamos nuestro mapa con la localización
